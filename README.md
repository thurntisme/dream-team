# Dream Team

A minimal football team builder webapp.

## Features
- User authentication (login/register)
- Club name setup
- Team formation selection (4-4-2, 4-3-3, 3-5-2, 4-2-3-1)
- Interactive player selection (11 players)
- Team saving

## Tech Stack
- PHP (with SQLite)
- Tailwind CSS
- jQuery
- Lucide Icons

## Setup
1. Ensure PHP is installed with SQLite support
2. Run with PHP built-in server:
   ```bash
   php -S localhost:8000
   ```
3. Open browser to `http://localhost:8000`

## Usage
1. Register/Login
2. Enter your club name
3. Select formation
4. Choose 11 players from the list
5. Click players on field to remove them
6. Save your team
