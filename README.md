# Dream Team

A minimal football team builder webapp.

## Features

- User authentication (login/register)
- Club name setup
- Team formation selection (4-4-2, 4-3-3, 3-5-2, 4-2-3-1)
- Interactive player selection (11 players)
- Team saving

## Tech Stack

- PHP (with SQLite3)
- Tailwind CSS
- jQuery
- Lucide Icons

## Setup

1. Ensure PHP is installed with SQLite3 support
2. Run with PHP built-in server:
   ```bash
   php -S localhost:8000
   ```
3. Open browser to `http://localhost:8000`
4. Follow the installation wizard to configure and set up the database
5. Configure application name and database file in the installer
6. Create an admin user to complete setup

## Usage

1. Register/Login
2. Enter your club name
3. Select formation
4. Choose 11 players from the list
5. Click players on field to remove them
6. Save your team
